# Service Request Application

## Overview

A full-stack application to create and manage service requests.

## Technologies Used

- React
- Node.js
- Express
- JSON File Storage

## Features

- User Registration
- User Login
- Create Service Request
- View Service Requests
- Update Request Status
- Delete Request

## Backend Setup

```bash
cd backend
node server.js
```

Runs on [http://localhost:5000](http://localhost:5000)

## Frontend

Runs on [http://localhost:3000](http://localhost:3000)
