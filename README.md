# Service Request Application

## Overview

A full-stack application to create and manage service requests.

## Technologies Used

- React
- Node.js
- Express
- JSON File Storage

## Features

- User Registration
- User Login
- Create Service Request
- View Service Requests
- Update Request Status
- Delete Request

## Backend Setup

```bash
cd backend
node server.js
```

Backend runs on:

http://localhost:5000

## Frontend Setup

```bash
cd frontend
npm start
```

Frontend runs on:

http://localhost:3000
